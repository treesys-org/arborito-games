# Changelog: Arborito Games (Arcade catalog)

Browser cartridges only. The app loads this repo via jsDelivr; bump **`meta.json` `version`** when behaviour changes so players can confirm what loaded.

## Catalog

### Wrong Fruit `1.0.1`

- Question banner stays on screen for 7 seconds so players can read it.
- Fruit answer labels show the full option text (wider plate, auto-sized font).

### Catalog `1.0.0`

All cartridges ship as **`1.0.0`**.

### Wrong Fruit

- Canvas fruits/gnomes (no missing emoji glyphs); large question flash; soft read-time cue.
- Each new quiz replaces the board (no leftover answers).
- Stage intermission pauses the worm; single animation loop; correct answer always on the board.
- Memory/XP reports the challenge just answered; next lesson reloads quizzes.
